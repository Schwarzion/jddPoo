## Requirements

* Node 13.11.0 (version used here)
* Git
* Command Line Client

## Common setup

Clone the repo and install the dependencies.

```bash
git clone https://github.com/Schwarzion/jddPoo.git
cd jddPoo
```

```bash
npm install
```

## Launch app

```bash
npm start
```


## About the game

This is a simple dice game
